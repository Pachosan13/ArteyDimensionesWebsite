# Team Profile Images

This directory contains profile photos for Arte y Dimensiones team members.

## Image Specifications

For best results, please use the following specifications when uploading team member photos:

- **Format**: JPG or PNG
- **Dimensions**: 800x1000 pixels (4:5 aspect ratio)
- **File Size**: Maximum 500KB per image
- **Orientation**: Portrait
- **Background**: Professional, neutral background preferred
- **Quality**: High resolution, well-lit, professional photography
